# SimpleHex
A simplistic and easy-to-use hex color generator

## Installation
1) Download `SimpleHex.zip` from [this page](https://github.com/REM-Codes/SimpleHex/releases/latest)
2) Extract `SimpleHex.zip` into a folder
3) Launch `SimpleHex.exe`
